## Notes for Devs ##

For ESP32C3, use ESP32-C3 Dev Module with Arduino 2.0.17 Core for ESP32, uncomment MODULE_ESP32C3_ELRS_TX

For ESP8266, use Generic ESP8266 Module with Arduino 3.3.6 Core for ESP8266, uncomment MODULE_ESP82XX_ELRS_TX
