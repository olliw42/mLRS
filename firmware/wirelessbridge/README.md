# mLRS - Firmware - WirelessBidge #

This folder contains pre-compiled wireless-bridge firmware binaries.

mlrs-wireless-bridge-esp32c3.ino.bin
- date of creation: 04.12.2025
- sketch version: 13.10.2025

mlrs-wireless-bridge-esp8266.ino.bin
- date of creation: 03.12.2025
- sketch version: 13.10.2025

## Notes for Devs ##

For ESP32C3, use ESP32-C3 Dev Module with Arduino 2.0.17 Core for ESP32, uncomment MODULE_ESP32C3_ELRS_TX

For ESP8266, use Generic ESP8266 Module with Arduino 3.1.2 Core for ESP8266, uncomment MODULE_ESP82XX_ELRS_TX
